# react-tinacms-i18n

A plugin suite for managing i18n (Internationalized) and l10n (Localized) TinaCMS projects.

Adheres to the [POSIX ISO/IEC 15897](https://en.wikipedia.org/wiki/Locale_(computer_software)) standards.
